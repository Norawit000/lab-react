เป็นโปรแกรมทายคำศัพท์เกี่ยวกับสัตว์  โดย ให้ไฟล์ Hangman.js ดำเนินการทำงาน setstate เปรียบเทียบเงื่อนไข
ต่างๆ และไฟล์ App.js เป็น component หลัก ส่วน Hangman.css และ App.css เป็นไฟล์ กำหนด สีรูปแบบ font ต่างๆของไฟล์หลัก   
โดยผมได้ import รูปภาพ Hangman เข้ามาเพื่อ ให้เห็นสถานะของการตอบผิดถ้าตอบผิดครบจน คนที่รูปผูกคอตาย
คนที่เล่นก็จะแพ้ไปและต้องเริ่ม restart เล่นใหม่  หน้าจอเกม แสดง จำนวนครั้งของคำตอบที่ตอบได้  กับ ข้อความว่าให้ลอง
เดาชื่อสัตว์ตามจำนวนช่องว่างที่ให้มา ถ้าอยากเล่นใหม่ก็ให้กดปุ่ม restart

โดยไฟล์ Hangman.js  มีฟังก์ชันต่างๆดังนี้  

 1.constructor(props)
กำหนด props ต่างๆ โดยกำหนด state  การตอบผิด การเดาคำตอบ คำถามให้ randomword  การรับมือการเดา this.handleGuess
การ this.reset ค่า

 2.reset()  กำหนดการ setstate ต่างๆ

3.guessedWord()  สร้างการ split  ข้อความ  ใน _

4.handleGuess(evt)  ให้ใส่ข้อวความเพิ่มเข้าไปในช่องว่าง ตรวจสอบ noofwrong

5.generateKeypad() => สร้าง ปุ่มกด pad ขึ้นมา  โดยมีตัวอักษรให้กดตั้งแต่  a-z  ใช้ฟังก์ชั่น map  มาแสดงค่าข้อความ
โดยมีonClickให้รู้สถานะว่าเม้าได้ทำการโดนคลิ้ก 

6.render() => เปรียบเทียบค่าระหว่าง จำนวนครั้งที่ผิด กับ จำนวนครั้งสูงสุดที่
ผิดได้  ถ้าเล่นแล้วชนะจะแสดงข้อความ Congrats, You have won the Game แต่ถ้าเล่นแล้วแพ้หน้าจอจะเด้ง
ข้อความ Good try! Better Luck Next Time โดยไม่ว่า แพ้ หรือ ชนะ โปรแกรมก็จะ reset ค่าให้เล่นใหม่เหมือนเดิม
